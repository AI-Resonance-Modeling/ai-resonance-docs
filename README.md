# AI Resonance Docs

This repository contains the conceptual and technical documentation for  
**AI Resonance Modeling (ARM)** — a protocol for building human-centric,  
stable AI personas.

## Structure

- `whitepaper.md` – high-level overview of AI Resonance Modeling.
- `protocol/intro.md` – introduction to the AI Persona Protocol.
- `protocol/arm-core.md` – core concepts, data structures and flows.
- `protocol/examples.md` – examples of persona capsules and configurations.
- `api/reference.md` – API-style description for implementers.
- `legal/license.md` – licensing and usage notes.

At this stage the documents are work-in-progress and may change frequently.
